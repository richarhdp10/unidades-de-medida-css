Unidades de Medida en CSS

Las diferentes unidades de medida utilizadas en CSS son :

Unidades Absolutas (px, cm)

Esta caja tiene un ancho fijo de 400px y su texto usa una fuente de 14px. Estas medidas no cambian si el usuario hace zoom o cambia el tamaño de la pantalla.

Altura de 1 cm

Unidades Relativas a la Fuente (em)

El tamaño de esta fuente es de **1.5em**.

Unidades Relativas al Padre (%)

Este párrafo tiene un margen que ocupa el 5% del ancho de esta caja.

Unidades Viewport (vh, vw)

Este contenedor tiene una altura de 20vh y un ancho de 50vw.
